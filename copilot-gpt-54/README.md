# Lars Nyström Portfolio

A React + Vite single-page portfolio built with TypeScript, TailwindCSS, and react-router.

## Setup

1. Install dependencies with `npm install`.
2. Start the development server with `npm run dev`.
3. Create a production build with `npm run build`.
4. Preview the production build locally with `npm run preview`.

## Stack

- React
- Vite
- TypeScript
- TailwindCSS
- react-router-dom
- lucide-react
